### 141. Linked List Cycle

[Question](https://leetcode.com/problems/linked-list-cycle/)
| [Solution](https://leetcode.com/submissions/detail/553478765/)

```c++
class Solution {
public:
    bool hasCycle(ListNode *head) {
        ListNode* slow = head;
        ListNode* fast = head;
        
        while(fast != NULL && fast->next != NULL){
            slow = slow->next;
            fast = fast->next->next;
            if(slow == fast)
                return true;
        };
        
        return false;
    }
};
```
